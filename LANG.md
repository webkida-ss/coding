# 言語別実行方法

## Go
```
go run main.go
```

### デバッグ

ライブラリインストール
```
go get -u github.com/derekparker/delve/cmd/dlv
```

## Python
```
python main.py
```

## Kotlin
スクリプト
```
kotlin main.kts
```

## TypeScript
スクリプト
```
ts-node main.ts
```

## Rust
スクリプト
```
cargo script main.rs
```

## PHP
スクリプト
```
php main.php
```
